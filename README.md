# Liar's Shot 🔫

Un jeu de cartes du Menteur avec mécanique de roulette russe, inspiré de Liar's Bar.

## Concept
- 4 joueurs autour d'une table de poker dans un saloon western
- À chaque round, une carte est annoncée (Roi, Dame ou Valet)
- Chaque joueur doit poser 1 à 3 cartes de ce type... ou bluffer !
- Les **As ★** sont des jokers (comptent comme la carte annoncée)
- Si quelqu'un appelle "Menteur !" et a raison → le menteur joue à la roulette russe
- Si l'accusation est fausse → c'est l'accusateur qui tire !
- Revolver à 6 chambres, 1 balle. Dernier survivant gagne.

## Fonctionnalités
- 🎨 Pixel art western avec décor de bar (bouteilles, affiches WANTED, lampes)
- 🤠 Personnages cowboys détaillés en SVG
- 🔫 Animation de roulette russe avec barillet qui tourne
- 🃏 Indicateur visuel clair du nombre de cartes posées
- ⏱ Timer configurable (10/15/20/30s)
- 😏 Système d'emotes rapides
- 🤖 3 bots IA avec comportements variés

## Stack technique
- React (JSX) - composant unique
- Animations CSS + SVG
- Pas de dépendances externes (sauf Google Fonts)
